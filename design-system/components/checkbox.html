Checkbox

Checkboxes let the user select one or more options from a list. They can also be used to toggle a particular option on or off.

Design guidelines

Checkbox or radio button?

A set of checkboxes should be used to select as many options as desired (or none). A set of radio buttons should be used to select only a single option from a list of mutually exclusive options.

Checkbox or toggle switch?

Checkboxes are best used for communicating selection while toggle switches are best used for activation (e.g. on/off states). Checkboxes, unlike switches, can have an error state.

Always use a checkbox when the user has to perform extra steps for changes to take effect.

Checkboxes should be positioned before the label

Accessibility

Always associate the label with the input

When a user clicks on the input’s label text, the checkbox should be activated. To achieve this, an id="example" should be added to the checkbox <input>. Then, this ID will be used as a for="example" attribute for the <label> element.

Multiple answers or selections

By default, there is an HTML element that makes multiple selections more accessible to screen readers.

The <fieldset> element provides a container for related form controls, and the <legend> element acts as a heading to identify the group. This will highlight the fact that it's a group and there is a question related to it with multiple answers.
