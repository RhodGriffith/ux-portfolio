Usage guidance

When to use this component

Only use this component when you have identified a need to quickly scan large sections of content. By hiding most of the content, users can spend their time more efficiently focused on the topics that are relevant to them.

Accordions are not a suitable replacement for well formatted plain text. If a user requires all of the information on the page to continue with their tasks, it should be visible not hidden inside an accordion.

Ensure both the icon and title area triggering expansion

Don't force users to desperately try and hit a small icon, some people have tremors which don't allow for fine muscle control.

Users are used to collapsing and expanding accordions by tapping on the content area within the bars. The accordion hover state also indicates the whole bar is clickable.

Arrows must face the way content expands

Arrows should face down to indicate content will expand downwards and face upwards to indicate when the content will collapse upwards.

Arrows pointing to the right should only be used to indicate and onward journey to another page.

Don't automatically collapse accordion items

It is not recommended that expanding one accordion item results in collapsing another. If a user has opened an item, assume they want it to remain open. This allows a user to compare open items if needed.

Another consideration is content on screen will start to jump around when you start closing accordions programatically.

For example, if a user selects item two, they would expect the accordion to open from that point on the screen. As item 1 would close at the same time, the newly opened accordion won't be in the same place.

Accessibility

Considerations for assistive technologies

Keyboard accessibility is one of the most important aspects of web accessibility. Many users with motor disabilities rely on a keyboard. Blind users also typically use a keyboard for navigation.

Assistive technologies users must be:

  • able to use the tab key on a keyboard to access each accordion item
  • able to use the return key or space bar to expand an collapse the accordion item
  • provided with a visual indicator of the element that currently has keyboard focus

The toggle button's state must communicated to users of assistive technologies with aria-expanded
